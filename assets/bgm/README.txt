Background Music (BGM) for HistoryShorts
==========================================

Place a file named "bgm.mp3" in this folder to enable background music in all videos.

The assembler mixes it at 12% volume (quiet, under the voiceover) and loops it automatically.

RECOMMENDED FREE SOURCES:
--------------------------
1. Pixabay Music (no attribution required):
   https://pixabay.com/music/search/epic%20history/
   https://pixabay.com/music/search/dramatic%20cinematic/
   https://pixabay.com/music/search/mysterious%20ancient/

2. Kevin MacLeod — Incompetech (CC BY 4.0, attribution in description):
   https://incompetech.com/music/royalty-free/music.html
   Good tracks: "Ossuary 6 - Air", "Investigations", "Dramatic Presentation"

3. YouTube Audio Library (free for YouTube use):
   https://studio.youtube.com/channel/music

GOOD STYLES FOR HISTORY SHORTS:
---------------------------------
- Epic / Cinematic (strings + percussion)
- Dark Ambient / Mysterious
- Ancient World / Middle Eastern
- Documentary / Investigative

HOW TO USE:
-----------
  1. Download any MP3 from the above sources
  2. Rename it to "bgm.mp3"
  3. Put it in this folder (assets/bgm/bgm.mp3)
  4. Run the pipeline — BGM will be mixed in automatically

Or run the auto-downloader from the project root:
  python download_bgm.py
